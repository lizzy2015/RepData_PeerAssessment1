2. Fill in all of the missing values

impute.mean=function(x) {
  z=mean(x, na.rm=T)
  x[is.na(x)]=z
  return(x)
}

3. Create a new dataset which is equal to the original dataset but with the missing data filled in

da.new=da
da.new$steps=impute.mean(da.new$steps)
